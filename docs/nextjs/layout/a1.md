# page

| 文件            | 作用           |
| --------------- | -------------- |
| `page.tsx`      | 页面           |
| `layout.tsx`    | 布局（可嵌套） |
| `loading.tsx`   | 加载中         |
| `error.tsx`     | 错误处理       |
| `not-found.tsx` | 404            |

## layout

```jsx
export default function Layout({ children }) {
  return (
    <html lang="en">
      <head>
        <title>My App</title>
      </head>
      <body>
        <header>Header</header>
        {children}
      </body>
    </html>
  );
}
```


## 1️⃣ loading.tsx（自动生效）

```txt
app/blog/loading.tsx
```

```tsx
export default function Loading() {
  return <p>加载中...</p>
}
```

## 2️⃣ error.tsx（必须是 Client）

```tsx
"use client";

export default function Error({ error, reset }) {
  return (
    <>
      <p>出错了：{error.message}</p>
      <button onClick={reset}>重试</button>
    </>
  )
}
```

# 基础路由（静态页面）

```
app/
  page.tsx        → /
  about/
    page.tsx      → /about
  contact/
    page.tsx      → /contact
```

# 三、动态路由（核心高频）

## 1️⃣ 单参数路由 `[id]`

```txt
app/
  posts/
    [id]/
      page.tsx     → /posts/123
```

```tsx
// app/posts/[id]/page.tsx
export default function PostPage({ params }: { params: { id: string } }) {
  return <h1>文章 ID：{params.id}</h1>;
}
```
## 多参数
```
app/blog/[category]/[slug]/page.tsx
```



## 2️⃣ 多参数 `[...slug]`（常用于分类）

```txt
app/docs/[...slug]/page.tsx
```

访问：

- `/docs/a`
- `/docs/a/b`
- `/docs/a/b/c`

```tsx
export default function DocsPage({ params }: { params: { slug: string[] } }) {
  return <div>{params.slug.join(" / ")}</div>;
}
```


## 3️⃣ 可选参数 `[[...slug]]`

```txt
app/docs/[[...slug]]/page.tsx
```

| URL       | params.slug |
| --------- | ----------- |
| `/docs`   | `undefined` |
| `/docs/a` | `["a"]`     |
